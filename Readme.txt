# My Book App (Demo Version)

My Book App is a simple web application for managing and reviewing books. Please note that this is a demo version, and the server-side functionality is not fully implemented. The data is hardcoded for demonstration and overview purposes.

## Features

- View a list of books.
- Add new books.
- View book details.
- Rate books.
- Write reviews for books.
- View user profiles.

## Installation

To run the My Book App (Demo Version) on your local machine, you need to have Node.js and npm (Node Package Manager) installed. If you don't have them installed, you can download and install them from [https://nodejs.org/](https://nodejs.org/).

Follow these steps to get the app up and running:

1. Unzip the myBook folder to your local machine
2. Change the directory to the project folder and install npm (Make sure you have Installed the latest Node.js version)
3. Install the project dependencies using npm inside the client folder:
npm install
4. Start the development server:
npm start
5. Open your web browser and visit [http://localhost:3000/]

## Usage

- Home Page: The default page displaying a welcome message and brief instructions.
- Book List: View a list of available books and click on a book title to view its details.
- Book Details: View details of a selected book, rate the book, and write a review.
- Add Book: Add a new book to the list.
- User Profile: View your user profile (hardcoded for demonstration purposes).
- Rating: Rate a book using a 5-star rating system.
- Review: Write and submit a review for a book.
